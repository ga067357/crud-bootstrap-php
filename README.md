# Aircraft Management CRUD System

## Overview
This project is an aircraft management system designed to manage aircraft information, users, and customer data through a complete CRUD (Create, Read, Update, Delete) interface. The system also features user authentication and customer management capabilities.

## Features
- **User Authentication:** Secure sign-in and registration functionality for users.
- **CRUD Operations:** Manage aircraft and customer data efficiently.
- **Responsive Design:** Built with Bootstrap for a fluid, mobile-first user interface.
- **Database Integration:** Utilizes a MySQL database for storing aircraft and user information.

## Technologies Used
- PHP
- MySQL
- Bootstrap
- HTML/CSS
- JavaScript

## Installation
1. Clone the repository:
    ```
    git clone https://github.com/ga067357/crud-bootstrap-php.git
    ```
2. Navigate to the project directory:
    ```
    cd crud-bootstrap-php
    ```
3. Import the `database.sql` file into your MySQL database.
4. Configure your database connection in the `config.php` file.
5. Run the application on your preferred PHP server.

## Usage
- Access the application via your browser at `http://localhost/crud-bootstrap-php`
- Register a new user or sign in if you have an account.
- Use the navigation to manage aircraft and customer records.
